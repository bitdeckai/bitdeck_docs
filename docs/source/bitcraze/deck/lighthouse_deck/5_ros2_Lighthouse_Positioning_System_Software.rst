介绍lighthouse定位系统相关软件
灯塔定位系统是我们在 Crazyflie 2.X 上最新的定位方法。 这里使用的是 Lighthouse 基站，它们发射红外激光扫描，由我们的 Lighthouse 定位平台检测到。 这使得crazyflie能够（就像使用 Loco 定位系统一样）计算自己的位置。
