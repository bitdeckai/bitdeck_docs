
ROS2与Lighthouse定位系统-硬件
============================

    介绍：
    -----
    本教程介绍lighthouse定位系统的硬件配置和硬件部件清单
    'Crazyflie无人机集群套件,采购地址 
    <https://item.taobao.com/item.htm?ft=t&id=858112103307&spm=a21dvs.23580594.0.0.52de2c1bg17fUu>'_

    1机灯塔定位套件
    ----------------
    1 x Crazyflie 2.1无人机
    1 x lighthouse Deck定位甲板 
    1  x Crazyradio PA 数传
    2  x  lighthouse v2定位器
    2  x  lighthouse v2定位器支架

    3机灯塔定位套件
    ----------------
    3 x Crazyflie 2.1无人机
    3 x lighthouse Deck定位甲板 
    1  x Crazyradio PA 数传
    2  x  lighthouse v2定位器
    2  x  lighthouse v2定位器支架

    6机灯塔定位套件
    ----------------
    6 x Crazyflie 2.1无人机
    6 x lighthouse Deck定位甲板 
    3  x Crazyradio PA 数传
    4  x  lighthouse v2定位器
    4  x  lighthouse v2定位器支架

    10机灯塔定位套件
    ----------------
    10 x Crazyflie 2.1无人机
    10 x lighthouse Deck定位甲板 
    3  x Crazyradio PA 数传
    4  x  lighthouse v2定位器
    4  x  lighthouse v2定位器支架

    lighthouse v1和v2 参数


    
========  ===========  ===========  ================================
特性      灯塔 V1      灯塔 V2      备注
========  ===========  ===========  ================================
范围        ~6米        6米          V1范围取决于环境，V2设计范围为6米
定位频率  30Hz(2基站)  ~50Hz        V2频率与基站数量无关
基站数量   1-2         1-4          V2硬件最多支持16个，可在固件中配置
水平视角   120°        150°         -
垂直视角   120°        110°         -
========  ===========  ===========  ================================
